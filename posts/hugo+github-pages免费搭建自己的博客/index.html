hugo+github pages免费搭建自己的博客

Posted on 2025-03-27 in 技术 • 236 words • 2 minute read
Tags: 教程, github, hugo, 博客

GitHub Pages 可以用来展示一些开源项目、主持博客甚或分享您的简历。Hugo 是 Go 编写的静态网站生成器，速度快，易用，可配置。通过hugo生成静态网站部署到github pages就可以得到我们的私人博客。

一、环境准备

本文使用的是windows 10,linux和macos操作类似，可查看hugo官方文档。

1.Git下载

（1）前往【Git官网】，下载安装程序

（2）一直点下一步，默认安装即可

2.Hugo下载

前往【Hugo Github Tags】，选择对应版本下载，下载后解压即可

Windows下载版本：hugo_extended_xxxxx_windows_amd64.zip

二、本地搭建博客

1.创建博客

（1）在hugo.exe所在文件夹的地址栏敲打cmd，然后Enter唤起命令行

(2)输入命令创建hugo网站文件

hugo new site xxxx

(3)切换目录，并把hugo.exe复制到刚生成的文件夹中

cd xxxx

2.配置hugo主题

（1）到【Hugo Themes】，挑选下载主题

（2）将下载好的主题解压，主题目录放到/themes文件夹中

（3）删除主文件夹中的hugo.toml和content文件夹

(4)将exampleSite样例数据中的 Content文件夹 和 hugo.yaml（或config.yaml） 复制到主文件夹中

（5）修改 hugo.yaml（或config.yaml） 中的 theme，将他修改为跟主题文件夹同名

（6）启动服务，在本机试访问

hugo server -D    
http://localhost:1313

（7）修改主题配置

hugo.yaml（或config.yaml）和主题文件夹下的layouts文件夹等

三、Github部署

1.常规部署

（1）前往【Github官网】，创建仓库 {github用户名}.github.io

（2）在仓库的Setting -> Pages Deploy from branch,Branch选择main-> save

（3）回到hugo主文件夹中，cmd执行命令hugo -D，会生成 public 静态资源文件夹

（4）在 public 执行以下命令上传到github仓库上，第一次上传可能需要输入账号密码

git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin {你的github仓库地址}
git push -u origin main

（5）上传成功后，即可访问博客： https://{github用户名}.github.io

2.Github Action自动部署

（1）Github创建一个新的仓库，用于存放Hugo的主文件

（2）前往Setttings -> Developer Settings -> Personal access tokens，创建一个token(classic)，取名为TOKEN

（3）token选择永不过期，并勾选 repo 和 workflow 选项

（4）为保证安全，将生成的token，保存的仓库的变量中，前往Settings -> Secrets and variables -> Actions中设置

（5）在Actions中New workflows -> set up a workflow yourself新建一个hugo-deploy.yml文件，内容如下：

name: deploy

# 代码提交到main分支时触发github action
on:
  push:
branches:
  - main

jobs:
  deploy:
runs-on: ubuntu-latest
steps:
- name: Checkout
  uses: actions/checkout@v4
  with:
  fetch-depth: 0

- name: Setup Hugo
  uses: peaceiris/actions-hugo@v3
  with:
  hugo-version: "latest"
  extended: true

- name: Build Web
  run: hugo -D

- name: Deploy Web
  uses: peaceiris/actions-gh-pages@v4
  with:
  PERSONAL_TOKEN: ${{ secrets.TOKEN }}
  #TOKEN为第（2）步取的名字
  EXTERNAL_REPOSITORY: 你的github名/你的仓库名
  PUBLISH_BRANCH: main
  PUBLISH_DIR: ./public
  commit_message: auto deploy

想了解更多可查看【Github Action文档】和【Hugo文档】

（6）在hugo主文件创建.gitignore文件，来避免提交不必要的文件

# 自动生成的文件
public
resources
.hugo_build.lock

# hugo命令
hugo.exe

（7）将hugo的主文件上传到仓库，上传成功后会触发Github Action，来自动部署你的静态页面

git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin {你的github仓库地址}
git push -u origin main

(8)上传博客 将写好的.md格式博客放到hugo主文件夹的content\posts下，执行：

 git add .
 git commit -m "update"
 git push

至此，博客搭建完成，好好写作吧。
